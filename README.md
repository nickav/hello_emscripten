# hello_emscripten

## Setup

First, install `emcc` and add it to your path: https://emscripten.org/docs/getting_started/downloads.html

## Build

Run:

```sh
./build.sh
```

Then open:

- http://localhost:8080/sdl.html
- http://localhost:8080/sdl2_webgpu.html
- http://localhost:8080/webgl.html
- http://localhost:8080/webgpu.html
